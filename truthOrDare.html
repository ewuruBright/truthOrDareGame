    Game Rules

    About the Game Rules / Guide

    The Game rules will guide you on How to play the game without any confusion.

    How to Add Dare

    If you want to add new Dare on your game you have to write in your input a dare text and make sure that the text you wrote in your input does not match the on you have on the list, if so, you will see an error message that says this Item Already exist, and note that without you writting anything in your input and when you click submit it will not add a new Dare to your Dare List, instead you will see an error message, if the text you type in your input does not match the Dare list you will see a successful message that said, successful added to the Dare list.

    How to Add Truth

    If you want to add new Truth on your game you have to write in your input a truth text and make sure that the text you wrote in your input does not match the on you have on the list, if so, you will see an error message that says this Item Already exist, and note that without you writting anything in your input and when you click submit it will not add a new Truth to your Truth List, instead you will see an error message, if the text you type in your input does not match the Truth list you will see a successful message that said, successful added to the Truth list.

    Deleting an Item in the Dare List

    If you want to cancel some item in your Dare list you can quickly click on the Add Dare button to go to the Dare List item and you will see a red button after the dare text by clicking it will delete that particular item that come's with the button, if the Dare list is empty you will see a message saying please insert a Dare List and Note: Without any List in the Truth Item or the Dare Item,the game will not be able to start, there must be at least one item in the Truth and Dare list to start the game.

    Deleting an Item in the Truth List

    If you want to cancel some item in your Truth list you can quickly click on the Add Truth button to go to the Truth List item and you will see a red button after the Truth text by clicking it will delete that particular item that come's with the button, if the Truth list is empty you will see a message saying please insert a Truth List and Note: Without any List in the Truth Item and the Dare Item, the game will not be able to start, there must be at least one item in the Truth and Dare list to start the game.

    How to Add Numbers of Players

    For you to add the number of players, you need to start the game by click on the start Game, after that you will see a new graphic layout / UI(User Interface), you will see an input with a number of 2, you can edit it by typing a number key with your keyboard Note: (It only accept Number, and the minimum Number you can put for the number of players is 2 and the Maximum number is 20). So after that you can click on the add button, when the add button is been clicked you will see an input been displayed, you have to fill all the input with the players name for you to enable the Let's Play button (to progress in playing the game or move to a new graphic layout / UI(User Interface) Note: No Players Name should repeat twice else you will see an erro message which will tell you player name that repeat twice and the game will not be able to proceed to the next stage.

    How will i know if it is my Turn

    After Clicking the Let's play button you will see a text that said the player name turn which is above the Truth and Dare button it said It's Player 1 turn then you click on any of the Truth and Dare button after that, you will move to be next graphic layout / UI(User Interface) which tell you what to do or say. Note: The players name come's out at random, so a player name can come out more that three times.

    What Does it mean for me to click the Completed button

    You clicking the completed button means you have completed what they told you to do, and after that it will move to the next-random player turn Note: This means that you are not involve in any punishment or Paying of money

    What Does it mean for me to click the Forget button

    You clicking the Forget button means you have not completed what they told you to do, and this will lead you to the punishment graphic layout / UI(User Interface), you will see the punishment message after completing the punishment task click on the Done button to go to next-random player turn Note: This means that you are involved in the punishment and Paying of money, this is a must.
